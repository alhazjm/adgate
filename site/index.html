Try the claims gate

the deterministic rule layer of a gated ad-creative pipeline · runs entirely in your browser, no model call, no cost · full run report · source on GitHub

Paste any ad hook and caption for a baby-care product. Ten do-not-say rules run first in the real pipeline — free, before any model is called — and each rule traces to citable evidence: the brand’s own published substantiation footnotes, Singapore’s Code of Advertising Practice, regulator precedent, and real customer complaints in public reviews. In the full pipeline, survivors then face an LLM brand-voice judge, and refused variants go to a bounded repair agent whose rewrites re-enter this same gate. This page runs the rule layer exactly as shipped.
